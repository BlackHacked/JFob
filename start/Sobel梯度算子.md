Sobel梯度算子
?
是图像处理中常用的一种边缘检测算子。它通过计算图像在水平方向和垂直方向的近似梯度,来检测图像中的边缘。具体而言,Sobel算子使用两个3×3的卷积核(滤波器),一个用于计算图像的水平方向梯度近似值,一个用于计算垂直方向梯度近似值。这两个 kernel 如下:Gx(求水平方向梯度) Gy(求垂直方向梯度)  
-1 0 +1       -1 -2 -1  
-2 0 +2        0  0  0   
-1 0 +1        +1 +2 +1
这两个核与图像进行卷积运算,从而得到图像在水平方向和垂直方向的梯度近似图。然后根据这两个梯度近似图,可以计算得到图像各个位置的梯度幅值(用它表示边缘强度)和梯度方向。通常情况下,梯度幅值较大的位置对应于图像的边缘。所以,在这段论文内容中,使用Sobel梯度算子的目的是要检测和增强图像中地面表面对应的边缘,进而通过阈值分割将地面像素从图像中分离出来。
1. Sobel算子是一种比较简单的边缘检测方法,它通过计算图像在水平和垂直两个方向上的近似梯度来检测边缘。更复杂的边缘检测方法还有Canny算子、LoG算子等
2. Sobel算子使用的两个3×3内核只能检测图像中0°,45°,90°和135°四个方向上的边缘。对于其它方向的边缘不太敏感。所以它对轮廓的提取不是很理想。
3. Sobel算子计算的梯度近似值使用的只是中心像素及其周围8个像素的值,所以它对图像噪声比较敏感。通常需要与Gaussian平滑等预处理或者后处理结合使用。
4. Sobel算子计算的梯度近似值的准确性还取决于内核选择。不同的内核会导致不同的梯度估计值。选择内核需要根据图像类型和噪声特点综合判断。
5. 在图像边缘处,Sobel算子的梯度幅值会比较大。但在自然图像的纹理区域,也可能产生较大梯度,需要设定一个适当的阈值来排除这些区域。阈值设置也需要根据图像特点实际调整。
6. Sobel算子的计算量比较小,处理速度较快,适用于实时应用。但对图像边缘的定位准确性稍差。在需要高精度边缘提取的应用中,更多使用Canny等算法。
